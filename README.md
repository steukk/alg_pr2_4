# Анализ алгоритма суммы цифр числа

## Python

### Алгоритм рекурсивного вычисления суммы цифр:

**Базовый случай**: Если число меньше 10 (содержит одну цифру), функция возвращает само число как результат.

**Рекурсивный случай**: Для чисел с двумя и более цифрами алгоритм:
- Извлекает последнюю цифру числа с помощью операции `n % 10`
- Рекурсивно вызывает себя для оставшейся части числа `n // 10` (целочисленное деление)
- Суммирует последнюю цифру с результатом рекурсивного вызова

**Процесс повторяется** до достижения базового случая, после чего результаты складываются по цепочке возвратов из рекурсивных вызовов.

### Временная сложность: O(d)

**Почему O(d)**: Алгоритм выполняет ровно один рекурсивный вызов для каждой цифры числа, где d - количество цифр. Каждая операция (остаток от деления и целочисленное деление) выполняется за константное время O(1), поэтому общее количество операций пропорционально количеству цифр в числе.

## Общий пример выполнения для числа 1234:

```
sum_digits(1234)
= 4 + sum_digits(123)
= 4 + (3 + sum_digits(12))
= 4 + (3 + (2 + sum_digits(1)))
= 4 + (3 + (2 + 1))
= 10
```

Каждый рекурсивный вызов обрабатывает одну цифру, пока не останется одна цифра (базовый случай).
